# Плейбук 04 — Profile Optimize (оптимизация профилей)

> **Когда запускать:** после подтверждения позиционирования (Ритуал B), и периодически (раз в месяц или при смене фокуса).
> **Язык:** RU для.hh.ru/Habr; EN для LinkedIn/Behance/личного сайта.
> **Режим:** human-in-the-loop — даёте переписанные секции, пользователь вставляет сам.
> **Входы:** `profile/positioning.md`, ссылки на текущие профили (из `intake-answers.md` → блок «Материалы»).

---

## Цель

Сделать пользователя **находимым** под целевые запросы рекрутеров и согласовать все профили с подтверждённым позиционированием. Это «сигнал»-петля.

## Платформы (по приоритету)

| Платформа | Язык | Фокус |
|---|---|---|
| LinkedIn | EN | Глобальный inbound; headline=позиционирование, не должность |
| hh.ru | RU | Домашний рынок; keyword-плотность, «Желаемая должность» с синонимами, портфолио |
| Habr Карьера | RU | Дизайн-культурные работодатели; кураторный профиль |
| Behance / Dribbble | EN/RU | Дизайн-native inbound; избранные кейсы |
| Личный сайт | EN/RU | SEO, сборка всего в одну витрину |

## Выход

`output/profile-audit-YYYY-MM-DD.md` (RU для hh/Habr, EN-секции для LinkedIn/Behance):
```md
# Аудит и правки профилей — YYYY-MM-DD

## LinkedIn (EN)
- Текущий headline → предложенный (позиционирование + ключевые слова)
- About: переписанный черновик
- Featured: какие 3 кейса поставить и почему
- Skills/Keywords: список для плотности

## hh.ru (RU)
- «Желаемая должность»: варианты с синонимами
- Сопроводительное: шаблон (большинство пропускают = ваше преимущество)
- Готовность к удалёнке/релокации: настройки

## Habr Карьера (RU) ...
## Behance ... 
```

## Правила

1. **Headline = позиционирование, не должность.** «Product Design Lead, специализируюсь на …» сильнее, чем просто «Senior Designer». (см. раздел 1 `positioning.md`)
2. **Keyword-плотность** — рекрутеры ищут по ключевым словам; обеспечьте вхождение терминов из позиционирования.
3. **3 опорных кейса** (раздел 11) — на витрину везде.
4. **Согласованность каналов** — одна история на всех платформах.
5. **Черновики только** — не редактируете профили сами; отдаёте тексты пользователю.
6. Никогда не просите и не используйте пароли/доступы к аккаунтам.
