# market-config — Россия (по умолчанию)

> Конфигурация рынка для агента (используется плейбуком `01-market-sense`).
> RU — по умолчанию (наш рабочий пример). Структура изолирована, чтобы пользователь
> мог заменить содержимое на другой рынок без правки остального кода.
> **Статус:** заглушка (P0). Расширяется в P4.

## Источники для market-sense
| Источник | Тип | Примечание |
|---|---|---|
| getmatch.ru | доска (кураторная, дизайн/tech) | лучший сигнал/шум для senior-уровня |
| career.habr.com | доска + зарплаты | фильтр Lead/Senior; зарплатные цифры gated |
| hh.ru | доска (доминирующая) | объём; keyword-поиск; отклики |
| vc.ru | контент | контент-маркетинг для работодателей, не вакансии |
| LinkedIn (EN) | глобальный inbound | в РФ официально недоступен (VPN); ToS caveat |

## Методология зарплатного якоря
- НЕ опираться на один агрегатор (малые выборки, смещения).
- Агрегировать: hh.ru-аналитика + Habr (фильтр Lead) + getmatch.
- Учитывать **уровень** пользователя (Lead/Senior выше средних цифр по профессии).
- Читать `profile/region` пользователя; по умолчанию — Санкт-Петербург.
- Ориентир (generic product designer, 2026, dreamjob.ru): средн. ~200 000 ₽/мес на руки; диапазон 170–230 тыс.; top Domclik/T-Банк/Яндекс. Это **пол**, не цель для Lead.

## Работодатели (ориентир, уточнять live)
Продуктовые: Яндекс, VK, Ozon, Avito, Т-Банк (часто remote). Растущий сегмент healthtech/edtech.

## Локализация по умолчанию
- Город: Санкт-Петербург (читать из `profile/region`).
- Валюта: RUB.
- Связь: культура Telegram-first (после первого контакта рекрутеры часто переходят в TG).

## Замена рынка
Чтобы адаптировать под другой рынок: замените таблицу источников, зарплатный якорь, список работодателей и локализацию. Остальные плейбуки остаются без изменений.
